# 📜 Açıklama
Terminale `npm install aoi.js@5.1.2` yazın. <br>
Discord Developer Portaldan botunuzdaki tüm intentleri açtığınızdan emin olun! <br>
Botunuzu çalıştırmak için `bot.js` dosyasına girin ve `token` kısmını doldurun.
